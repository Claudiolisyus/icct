# **App Name**: ICCT (Lagay Title Mga Supot)

## Core Features:

- Appointment Booking (Students): Students can book appointments online for SOG (summary of grades) and prospectus evaluation with date/time selection.
- Appointment Scheduling (Registrar): Registrar/Admin can properly organize, monitor schedules, and manage student appointments. Includes an admin dashboard showing all bookings and daily schedules.
- Search & Filtering: Admin can search and filter appointments by student, service, and date.
- Notifications/Reminders: Automated email/SMS notifications and reminders for upcoming appointments.
- Reporting and Export: Generate reports on appointment metrics and export data for analysis.
- User Authentication: Secure login/signup functionality for both students and administrators. Admin login: BigDaddyKenneth@gmail.com/Hamburger99
- AI-Powered Automated assistant: An LLM-based tool, with the ability to send personalized, scheduled email reminders and follow-ups based on booking status.

## Style Guidelines:

- Primary color: Light blue (#64B5F6) to align with ICCT's branding and create a calm, professional feel.
- Background color: Very light blue (#E3F2FD), a desaturated tint of the primary, to create a clean, spacious backdrop.
- Accent color: Light Indigo (#7986CB), to provide contrast to CTAs and actionable elements.
- Body and headline font: 'PT Sans', a modern sans-serif for readability and a touch of warmth.
- Use clear and recognizable icons for appointment types and actions. Inspired by the provided visual example, opt for outlined icons over filled ones.
- Clean and intuitive layout, drawing inspiration from the reference image's organized dashboards and calendar views. Prioritize user-friendliness to minimize navigation time.
- Subtle animations on button hovers and transitions to provide feedback and enhance user experience.
